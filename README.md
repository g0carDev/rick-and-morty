# Rick And Morty App

This is a simple PWA that uses the Rick and Morty API to show the characters and episodes of the series.

## Installation

Clone the repository and run the following commands:

```bash
npm install or yarn
npm start | yarn start
```

## Stack

- React (18.2.0)
- React Router (6.8.1)
- Redux Saga (1.2.2)
- Reduxjs Toolkit (1.9.2)
- MUI Material (5.11.9)
